# 제안 #04 · TWO-LOOP TOURNAMENT · 이중 루프 정원

> 매치 사이 20초의 "덱 정원"에서 카드를 물주고, 다듬고, 융합한다. 두 개의 루프가 서로를 먹여 살린다.

## 컨셉 (한 줄)

**살아있는 덱을 키우는 식물학자의 토너먼트** — 매치는 원작 그대로의 60~90초 자동 리빌 결투지만, 라운드 사이 15~25초 인터미션에 덱이 실제로 자라나는 정원이 된다. 좋은 정원 → 좋은 매치 → 더 많은 정원 액션의 이중 피드백.

## 톤 & 무드

- **아트**: 이끼 낀 브루탈리스트 콘크리트 위에 크롬 화분이 놓인 미래식 온실. 카드는 유리 표본실 안의 살아있는 식물 — 잎맥이 사이언 광섬유처럼 은은히 발광. 부드러운 실내 비. 젖은 콘크리트에 반사되는 네온.
- **사운드**: 물방울 톡톡, 가위질 스낏, 잎사귀 스라락, 융합 순간 낮은 종소리 챙~ 잔향. 인터미션 BGM은 앰비언트 신스 + 빗소리 로우파이. 매치 BGM은 미묘하게 빨라진 같은 트랙.
- **컬러 팔레트**:
  - `#0F5C3D` 딥 에메랄드 (배경 · 시그니처)
  - `#6B8E5A` 모스 그린 (UI 프레임 · 이끼)
  - `#B87333` 코퍼 (강조 · 화분 테두리)
  - `#F0E8D0` 소프트 크림 (텍스트 · 카드 이름)
  - `#3D3D3D` 매트 콘크리트 (베이스)
  - `#88F0FF` 소프트 사이언 글로우 (잎맥 · 파워)
  - `#D9B8FF` 옅은 라벤더 (융합 순간 스파클)
- **타이포**: 카드 이름 · 헤더 `Cormorant Garamond` (식물학 도감의 세리프), 마이크로/타이머 `Space Mono`, 파워 넘버 `Cormorant` 이탤릭 넓게.

## 코어 루프

### Loop A — MATCH (60~90초, 원작 그대로)
1. **0~5s**: 상대 파일 오픈, 방어 파워 표시. 어택 시작.
2. **5~55s**: 자동 리빌 tug-of-war. 카드 슬라이드 인 → 파워 합산 → 카드에 물방울/잎 잔재 이펙트 (Loop B에서 물 준 카드는 반짝임).
3. **55~75s**: 파일 넘어감. 벤치 정리. 승패 결산.
4. **75~90s**: 매치 리워드 표시 — 다음 인터미션에 쓸 "물방울(액션 포인트)" 3~5개 지급.

### Loop B — GARDEN INTERMISSION (15~25초, 신규)
1. **0~2s — 씬 트랜지션**: 카메라가 덱 걸이에서 뒤로 빠지며 크롬 화분들이 밑에서 솟아오름. 카드들이 부드럽게 호흡(scale 1.0 ↔ 1.03).
2. **2~18s — 정원 액션**: 하단에 3개 버튼 (💧 WATER / ✂️ PRUNE / 🌱 FUSE). 손가락으로 카드를 탭 후 액션 버튼.
   - **WATER (1 포인트)**: 카드 파워 +1 (해당 매치 한정 or 영구, 밸런싱 이슈).
   - **PRUNE (1 포인트)**: 덱에서 카드 1장 완전 제거 (원작 덱 제거의 프레임 변경).
   - **FUSE (2 포인트)**: 같은 이름 카드 2장 → 승급된 변종 1장. **FUSION BLOOM** 시그니처 이펙트.
3. **18~25s — 타이머 압박**: 남은 시간 3초 이하에서 카드 프레임이 붉게 깜빡. 자동으로 매치로 전환.

두 루프가 서로 리드미컬하게 붙어 있어서 세션 전체가 "숨 → 결투 → 숨 → 결투" 리듬이 됨.

## 씬 목록

### LOBBY — "THE GREENHOUSE GATE"
- 이끼 낀 콘크리트 문 앞. 크롬 화분 하나에 앤세스트럴 시드가 심겨 있음.
- 메인 CTA: 코퍼 링 안에 "ENTER". 서브 CTA는 SEED VAULT / CODEX / OPTIONS.
- 배경엔 소프트 레인, 좌하단에 이번 러닝 예정 상대 실루엣 프리뷰 3인.

### GARDEN — "THE INTERMISSION" *(시그니처 씬)*
- 세로 화면 상단에 큰 타이머 `0:18` (Space Mono).
- 중앙에 크롬 화분 6~8개가 2열 그리드. 각 화분에 카드가 식물처럼 세워져 은은히 호흡.
- 카드마다 이름(라틴 도감체 세리프) + 레벨 도트(●●○). 화분엔 코퍼 라벨.
- 하단에 액션 버튼 3개 + 물방울 포인트 잔량. 우측에 "FUSION PREVIEW" 미니 슬롯.

### MATCH — "THE DUEL"
- 원작 tug-of-war. 상단 상대 파일, 하단 내 파일.
- 카드 리빌 시 잎사귀 파티클 잔재. 물 준 카드는 리빌 순간 사이언 잎맥이 짧게 반짝.
- 중앙 파워 카운터는 코퍼 링 안 크림색 숫자. 벤치는 좌우 미니 화분 6칸.

### RESULT — "THE HERBARIUM"
- 매치 종료 후 승자의 카드가 유리 표본실 안에 압화(pressed flower)처럼 안착.
- 획득한 팬은 씨앗 봉투로 시각화. 우승 시 러닝 종료 → 앤세스트럴 시드 선택 화면.

## 메타 레이어

- **ANCESTRAL SEEDS**: 러닝 종료 시 획득한 팬 수에 따라 시드 1~2개 선택. 시드는 **다음 러닝 시작 덱에 영구 심겨진 변종 카드**를 제공. 예: "달빛 이끼 씨앗 → 밤 카드 파워 +2", "구리 뿌리 → 시작 물방울 +2".
- **HERBARIUM CODEX**: 러닝 중 만난 모든 카드/융합 조합이 도감에 등록. 도감 완성도에 따라 로비 배경에 식물이 늘어남 (레거시 성취).
- **WEATHER MODIFIER**: 매 라운드 배경 날씨가 랜덤 (맑음/비/안개). 날씨에 따라 정원 액션 코스트가 미세 변동 — 비오는 라운드엔 WATER 무료.

## 시그니처 모멘트

1. **FUSION BLOOM · 융합 개화**
   같은 이름 카드 2장을 두 손가락으로 동시에 탭 → 카드가 서로 끌려가며 겹침. 잎사귀 파티클이 원심으로 폭발 + 라벤더 스파클 + 낮은 종 챙~. 새 승급 카드가 소프트 크림 라이트를 뿜으며 화분에 착지. 화분 라벨이 코퍼 → 골드로 바뀜.

2. **CARDS BREATHE · 카드가 숨쉰다**
   인터미션 진입 순간, 모든 카드가 미묘한 scale 1.0 ↔ 1.03 사인파로 호흡 시작. 각 카드마다 위상이 살짝 다름 → 정원 전체가 살아있는 느낌. 사운드는 낮은 풍성한 톤이 카드마다 겹쳐 화음.

3. **RAIN ROUND · 비의 라운드**
   랜덤 라운드에 소프트 레인 시작 → 인터미션 WATER 액션 무료. 화면 전체에 물방울 파티클, 카드마다 물방울이 잎에 맺힘. 매치 중에도 비가 계속 오면서 특정 카드 파워 +1 (물에 반응하는 카드).

## 현 버전 대비 Tradeoffs

**얻는 것**
- 인터랙션 밀도 증가 → 자동 리빌만 보고 있는 지루함 해소.
- Ball×Pit / CloverPit 같은 이중 루프 감성 → 리텐션 강함.
- "내가 키운 덱" 애착 형성 → 러닝 실패도 정원 성장으로 회수됨.
- 톤이 명확 (Balatro 계열과 완전 차별화).

**잃는 것**
- 세션 길이 증가: 7라운드 × (매치 90s + 인터미션 25s) = ~13분. 원래 5~10분에서 벗어남.
- UX 복잡도 증가 → 튜토리얼 필수, 첫 러닝 진입장벽.
- Loop B 밸런싱 부담 (WATER/PRUNE/FUSE 비용 밸런싱).
- 시각적으로 "예뻐야" 성립 → 아트 리소스 최소치가 Balatro 안보다 큼.
