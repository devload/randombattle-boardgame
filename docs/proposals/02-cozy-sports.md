# 02 · COZY SPORTS LEAGUE — 작은 마을 파이 대회

## 컨셉 (한 줄)
동물 마을의 여름 축제 **파이 콘테스트**에서, 손수 고른 재료 카드로 심사위원 앞에 파이를 쌓아 올리는 따뜻한 1인 토너먼트.

## 톤 & 무드
- **아트**: 크레용·수채 스티커 감성, 손그림 테두리, 살짝 삐뚤빼뚤한 라벨 태그. 동물 셰프는 라쿤/토끼/곰/여우/오리 등 통통한 실루엣.
- **사운드**: 나무 스푼 톡톡, 파이 오븐 "띵!", 잔잔한 우쿨렐레 루프, 심사위원 리액션은 "음~♪" 짧은 보컬 샘플.
- **컬러 팔레트**
  - `#faf3e0` 크리미 버터 (배경)
  - `#a5c9a5` 세이지 그린 (프레임·리본)
  - `#ff7e6b` 웜 코랄 (하이라이트·심사평)
  - `#c5b28c` 소프트 브라운 (파이 크러스트·나무)
  - `#ffe6a3` 버터 옐로우 (별·강조)
  - `#5b4636` 딥 코코아 (텍스트)
- **금기**: 네온, 하드 섀도우, 각진 UI, 위협적 카피 ("배틀", "파괴" 대신 "심사", "한 조각").

## 코어 루프 (60~90초 한 판)
1. **0~5s** — 상대 셰프(예: 라쿤 셰프) 카드 페이드 인, "이번엔 어떤 파이를 낼까?" 말풍선.
2. **5~15s** — 상대가 먼저 리본(=플래그)에 파이 한 조각을 올림. 파이 스택 첫 층 등장.
3. **15~45s** — 내 턴. 재료 카드를 위→아래로 쓸어 파이 위에 올리기. "맛 미터"가 채워지며 심사위원 얼굴이 점점 밝아짐.
4. **45~60s** — 미터가 상대치를 넘는 순간 **GRAND JUDGE'S TASTE** 컷인. 심사위원 3인방(할머니 오리·소년 곰·고양이 평론가) 리액션 이모지 팝업.
5. **60~75s** — 리본이 내 쪽으로 넘어옴. 벤치의 동물들이 박수/하품(피로 누적).
6. **75~90s** — 팬(스코어) +N, 다음 매치 프리뷰. 벤치에 7번째 종이 앉으면 그 라운드 종료.

## 씬 목록

### Lobby — 마을 광장
- 상단에 오늘의 축제 배너, 하단에 파스텔 벤치.
- "오늘의 대회 시작하기" 큰 버튼 하나. 좌측엔 내 조리대 미리보기 카드.
- 계절 표시(여름 축제/가을 수확제)로 라이브한 느낌.

### DeckPhase — 시장 (Market)
- 좌판 3~4개에서 재료 카드 뽑기. 각 카드는 스티커 형태(딸기 잼, 솔방울, 라벤더 꿀 등).
- 하단에 "내 레시피북" — 지금까지 고른 카드 리스트가 파이 도식으로 시각화.
- 완성 확인 시 오븐 아이콘이 부풀며 다음 씬으로 전환.

### Match — 심사대 (핵심)
- 중앙에 **파이 스택**(플래그 파일), 좌우에 셰프 초상.
- 하단 카드 트레이에서 재료를 스와이프 업 → 파이 위에 얹힘.
- 우측에 벤치(6칸) — 지친 동물들이 앉아 응원 또는 꾸벅꾸벅.
- 이벤트 로그: "솔방울 잼은 완벽해요! 🥧", "라쿤 셰프가 시나몬을 추가!"

### Result — 시상식
- 리본(1등/2등/참가상)이 나풀나풀 내려옴. 팬 수만큼 마을 주민 이모지가 박수.
- "다음 대회로" / "조리대 꾸미기" / "레시피북 보기" 3버튼.

## 메타 레이어
- **조리대 데코**: 매치 승리 시 코인(도토리) 획득 → 앞치마, 파이 접시, 창가 화분 등 소소한 커스텀.
- **레시피북 언락**: 라운드 클리어마다 시그니처 카드 1장 해금 (예: "할머니의 블루베리 크럼블").
- **계절 이벤트**: 여름 축제(딸기), 가을 수확제(호박), 겨울 벽난로전(계피). 계절마다 심사위원 룩과 배경 교체.
- **친구 마을**: 리플레이 URL 공유 시 상대 셰프에 친구 이름/동물이 삽입되는 소셜 훅.

## 시그니처 모멘트
1. **GRAND JUDGE'S TASTE** — 파이 완성 순간 배경이 크림색으로 하얗게 뜨고, 심사위원 셋의 반응 이모지가 통통 튀며 리본이 반대편으로 팔랑팔랑 이동. 꽃잎 파티클 낙하 + 소프트 챠임.
2. **오븐 부풀기** — DeckPhase 종료 시 오븐이 부풀어 다음 라운드를 예고. 짧지만 만족스러운 트랜지션.
3. **벤치 꾸벅** — 7번째 종이 벤치에 앉는 순간 나머지 동물들이 하품하며 담요를 덮음. "패배"가 귀엽게 느껴지도록.

## 현 버전 대비 Tradeoffs
- **장점**: 진입장벽이 낮고 TikTok/릴스로 클립화하기 좋음. IP 회피(재료·요리는 저작권 자유). 여성/캐주얼 유저 확보 가능.
- **단점**: "배틀" 어휘를 전부 리라이팅해야 하므로 도메인 코어의 네이밍 리팩터 필요. 카드 아트 부담(스티커 감성은 프롭 수량이 많아 보임).
- **원작 정체성**: 경쟁·긴장감이 순화되어 하드코어 보드게이머에겐 "심심"할 수 있음. 대신 재플레이성은 데코·언락으로 보완.

## MVP 범위 (2주 빌드)
- 씬 4개(Lobby / Market / Match / Result) 정적 + 매치 로직 연결.
- 카드 아트는 이모지 + 파스텔 뱃지 조합으로 대체 (실아트 X).
- 심사위원 3인 리액션 컷인 1종만 완성(승리/패배 공용, 얼굴만 다름).
- 벤치 슬롯 6칸 + 7번째 앉기 애니메이션.
- 메타 언락은 "레시피 3장" 최소 구현. 조리대 데코는 아이콘 색만 바뀌는 수준.
- BGM 1트랙 + SFX 4종(카드 놓기, 파이 완성, 리본, 하품).
