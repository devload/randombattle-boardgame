# 06 · FARAWAY · SOLO MOBILE

> **"카드는 왼쪽에서 오른쪽으로 놓고, 점수는 오른쪽에서 왼쪽으로 셈한다"**
> 미래를 예측하며 현재를 배치하는 15분 판타지 타블로 빌더.

원작: **Faraway** — Corentin Lebrat & Johannes Goupy, 2023.
Kennerspiel des Jahres 2024 수상. Nominated Golden Geek Best Card Game.

---

## 컨셉 한 줄

**카드를 놓는 방향과 점수를 세는 방향이 반대인 15분 타블로 빌더** — 지금 놓는 카드는 미래의 스코어러가 될지, 미래의 스코어러를 위한 셋업이 될지 매 턴 결정.

## 원작 룰 요약

### 게임 흐름
- 각 플레이어는 지역 카드(1~68) 손패 3장으로 시작
- 8라운드 진행. 매 라운드:
  1. **동시에 카드 1장 선택** → 자신의 타블로에 왼쪽부터 순서대로 놓음
  2. **턴 순서 결정**: 방금 놓은 카드의 **숫자가 낮은 사람부터** 다음 액션
  3. **드로우**: 순서대로 (a) 손패 보충 또는 (b) 특별 카드(성소·Sanctuary) 획득
     - 낮은 숫자 = 먼저 뽑지만 지역 카드(3장 중 1장)만
     - 높은 숫자 = 나중에 뽑지만 성소 카드까지 볼 수 있음
- 8라운드 후 종료 → 점수 계산

### 핵심 트릭: **역방향 스코어링**
게임 끝나면 자신의 8장 타블로를 **오른쪽부터 (마지막 놓은 카드) 왼쪽으로 (첫 카드)** 스코어링:
- 각 카드에 스코어 조건 (예: "낮 아이콘 3개 필요")
- 조건은 **그 카드의 왼쪽에 있는 카드들** (= 그 카드보다 먼저 놓은 카드들)로 판정
- 즉 **마지막에 놓은 카드는 앞에 놓인 모든 카드의 아이콘**을 활용 가능
- 첫 번째 놓은 카드는 판정할 왼쪽 카드가 없어서 스스로 스코어 조건이 있어도 채울 게 없음

### 시그니처 텐션
- 미리 아이콘을 왼쪽에 쌓아둔다 → 오른쪽 스코어러 카드가 크게 터짐
- 근데 손패는 계속 바뀌고, 어떤 스코어러가 미래에 올지 모름
- 강한 스코어러(70점짜리)를 지금 왼쪽에 놓아버리면 조건이 안 채워져서 0점
- 아무 것도 안 놓고 아끼면 8라운드에 못 놓음

## 코어 루프 (한 라운드 60~90초)

1. **손패 검토 (10~20s)**: 손패 3장을 훑어봄. 각 카드의 스코어 조건·아이콘·숫자 확인.
2. **미래 예측 (20~30s)**: 지금까지 놓은 타블로에 어떤 아이콘이 있는지 확인. 다음 스코어러가 이걸 활용할지 계산.
3. **카드 놓기 (5s)**: 왼쪽 슬롯에 스와이프해서 놓음 (부드러운 툭!)
4. **턴 순서 계산 (5s)**: 봇들과 내가 놓은 숫자 비교. 애니메이션으로 순서 알림.
5. **드로우 (10~20s)**:
   - 낮은 숫자: 지역 3장 중 1장 선택
   - 높은 숫자: 성소 시장에서 선택
   - 성소는 즉시 발동 효과 · 스코어 배수 등 강력

**8라운드 후 → 스코어링 세리모니 (30~60s)**:
- 타블로가 오른쪽부터 하이라이트 되며 점수 계산
- 각 카드의 조건과 왼쪽 아이콘이 매칭되는 애니메이션
- 최종 팬(스코어) 발표

## 씬 목록

### LOBBY
- 여명의 언덕. 지평선 실루엣.
- CTA: "REGION 1 시작하기"
- 이전 러닝 스코어 배지

### MATCH — 여정 (Journey) ★
- **상단**: 타블로 8칸 (지금까지 놓은 카드가 좌→우 채워짐)
- **중단**: 손패 3장 (탭해서 선택 → 스와이프로 배치)
- **하단**: 성소 시장 미리보기 (턴 끝날 때 이걸 뽑을 수 있는지 여부)
- **좌측 사이드**: 현재 라운드 · 예상 스코어 힌트 (선택적 온오프 가능)

### SCORE CEREMONY (신규 · 시그니처)
- 8라운드 종료 후 씬 전환
- 타블로가 화면 상단으로 이동
- 오른쪽 카드부터 하이라이트 시작 → 왼쪽 아이콘 스캔 → 조건 매칭 애니메이션 → 점수 더해짐
- 배경 앰비언트 밝기 상승. 종소리.

### RESULT
- 최종 스코어 (Cinzel 큰 숫자)
- 러닝 히스토리 (내 최고 스코어)
- CTA: "새 여정" / "여정 리플레이"

## 메타 레이어

### 성소 컬렉션
- 러닝 종료 시 등장한 성소 카드 자동 도감 등록
- 도감 100% 채우기 목표 (원작에도 있는 요소)

### 여정 프리셋
- 각 러닝은 고유 seed. 스코어 공유 시 friends가 같은 seed로 도전 가능
- Daily challenge: 매일 하나의 지정 seed, 리더보드

### 지역 언락
- 특정 스코어 달성 시 새 지역 카드 세트 언락 (원작에 없는 우리만의 layer)
- 예: 100점 초과 → "안개 숲" 세트 언락, 새 아이콘 등장

## 시그니처 모멘트

### 1. 스코어링 세리모니 ★
8장 타블로가 오른쪽부터 하이라이트되며 진행되는 30초 씬. 각 카드가:
- 스포트라이트 받으며 스코어 조건 확대
- 왼쪽 카드들 아이콘 스캔 (실시간 카운트)
- 조건 충족 시 → 골드 링 뜨고 점수 카운트업
- 실패 시 → 회색 fade
- 총합이 상단에 큰 Cinzel 숫자로 누적

### 2. 미래 예언 하이라이트
카드 놓을 때, 놓기 전 프리뷰:
- "이 카드가 왼쪽 조건이 되는 미래 카드가 얼마나 될까?" 힌트
- 손패의 각 카드가 지금 놓는 카드의 스코어 조건에 도움 되는지 화살표 표시

### 3. 성소 등장
높은 숫자로 뽑기 순서 뒤로 밀렸을 때 → 성소 시장 등장. 유리 케이스에 진열된 카드들 반짝. 하나 픽업 시 신비 챠임.

## 현 프로젝트 (챌린저스) 대비 tradeoffs

**얻는 것**
- ✅ 압도적으로 **다른 감성** (사이버펑크 → 판타지 힐링)
- ✅ 원작 룰이 정말 elegant — remake 아름다움 살아남
- ✅ 세션 15분 → 모바일 완벽
- ✅ 솔로가 원작에 이미 있음 (봇 rule 그대로 이식)
- ✅ 스코어링 세리모니가 감정 이입 훅
- ✅ 미래 예측 요소가 챌린저스 자동 리빌보다 훨씬 능동적 (플레이어 결정 다수)
- ✅ IP 회피 상대적으로 수월 (지역·판타지는 창작 가능)

**잃는 것**
- ❌ 지금 챌린저스 프로젝트를 처음부터 다시 시작
- ❌ 아트 리소스가 훨씬 중요 (풍경 일러스트)
- ❌ 밸런싱 리소스 (68장 카드 조합 · 성소 · 스코어링 튜닝)
- ❌ 원작 팬이 감지 못하는 미묘한 룰 스킵이 있을 수 있음
- ❌ 세리모니 씬 개발 부담
