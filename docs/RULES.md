# Challengers! 원작 규칙 정리

> **면책**: 아래는 팬 리메이크 개발을 위한 정리입니다. 저작권은 Johannes Krenner & Markus Slawitscheck / 1 More Time Games / Z-Man Games에 있습니다.

---

## 게임 목표

- **7라운드 토너먼트** 진행
- 각 라운드마다 다른 상대와 매치
- **팬(fan) = 승점**. 매치 승리 시 트로피 획득, 트로피 뒷면의 팬 수가 최종 점수
- **팬 1·2위 두 명이 결승**, 결승 승자가 챔피언

**솔로/2인 룰:**
- 결승 없음. 7라운드 후 팬 최다가 승리
- 언제든 팬 격차가 **11 이상**이면 그 시점에서 즉시 승리

## 라운드 구조

각 라운드 = **덱 페이즈 → 매치 페이즈**

### 덱 페이즈 (다른 플레이어와 독립적, 자기 페이스)

1. **대진표 확인** → 필요하면 다른 파크로 자리 이동
2. **카드 드래프트:**
   - 현재 라운드에 허용된 **레벨 파일(A/B/C)** 에서 5장 드로우
   - **Pick 규칙:**
     - Level-A pile → 2장 픽
     - Level-B pile → 2장 픽 **또는** Level-C pile → 1장 픽
   - 안 뽑은 카드는 discard 컴파트먼트로
   - **한 번 리롤 가능**: 뽑은 5장 버리고 같은 파일에서 재드로우
3. **덱에서 카드 제거 (선택):**
   - 원하는 만큼 자기 덱에서 카드 제거 가능
   - Level 카드는 discard 컴파트먼트로, Starter(S) 카드는 박스로

**중요 파라미터:**
- Level A 카드: base power 최대 3
- Level B 카드: base power 최대 5
- Level C 카드: base power 최대 10

---

### 매치 페이즈 (상대와 1:1)

#### A. 시작

- 자기 덱 셔플 후 자기 앞에 face-down
- **매치 중에는 자기 카드를 볼 수 없고, 순서 재배열도 못 함** (효과가 허용하지 않는 한)
- **선공 결정:**
  - 라운드 1: 깃발 던지기
  - 라운드 2+: 트로피 번호(round number)가 더 큰 사람이 선공. 동률이면 깃발 던지기
- 선공자가 덱 톱카드 리빌 → 자기 파크 반쪽에 놓음 → 그 위에 깃발. 이 카드가 **flag possession** 상태

#### B. 어택 (Attack)

**상대가 flag possession 상태일 때, 나는 어택 차례.**

1. 내 덱에서 카드를 **한 장씩** 리빌
2. 리빌한 카드들의 **총 파워** = 첫 카드 base + power bonuses + 둘째 카드 base + bonuses + …
3. **내 총 파워가 상대 flag possession 카드의 총 파워와 같거나 크면** → 깃발 획득. 마지막 리빌 카드 위에 깃발
4. 필요한 만큼만 리빌. 초과 리빌 금지
5. 깃발 획득 시, 리빌했던 나머지 카드들은 새로 flag possession 된 카드 **아래로** 밀어넣음

#### C. 벤치 (Bench)

- 깃발을 뺏긴 상대는:
  - 방금 flag possession이었던 카드 + 그 아래의 슬라이드된 카드들을 **모두 벤치로**
  - 벤치는 오른편에 **6석**
  - **같은 이름 카드는 한 시트에 겹쳐서** 놓음 (한 시트 = 한 종류)
- 벤치 카드 효과는 벤치에 있는 동안 계속 활성

#### D. 매치 승리 조건 (둘 중 하나)

**I. 상대가 어택인데 파워 부족으로 flag possession 못 함**
- 상대가 덱을 다 뽑고도 내 flag possession을 못 넘김

**II. 상대가 7번째 다른 이름의 카드를 벤치에 놔야 함**
- 벤치는 6종류까지만. 7번째는 놓을 자리 없음 → 매치 패

#### E. 매치 종료 후

- 승자는 자기 파크의 **현재 라운드 트로피** 획득 → 대진표 옆에 둠 (뒷면=팬 수, 상대에게 숨김)
- 뒷 라운드 트로피가 팬 수가 더 많음
- 자기 덱 회수: 벤치 + 파크 + exhaust pile + 안 뽑은 카드 다 모아서
- 라운드 1~6 후: 다음 라운드 덱 페이즈로

## 카드 효과 시스템

### 효과 발동 시점 (키워드)

| 키워드 | 언제 발동 | 활성 기간 |
|-------|---------|----------|
| **키워드 없음** | 리빌 즉시 | 파워 보너스는 flag possession으로 유지 |
| **During the attack** | 리빌 시 | 그 어택 중에만. flag possession 시 비활성 |
| **From the bench** | 벤치 진입 즉시 | 벤치에 있는 동안. 같은 이름 스택 시 보너스 누적 |
| **In flag possession** | flag 획득 시 | flag 유지되는 동안 |
| **Flag loss** | flag 뺏길 때 | 1회. flag possession 효과보다 먼저 발동 |
| **When picked** (Outer Space 세트) | 덱 페이즈에서 픽할 때 | 1회 |

### 기타

- 리빌 첫 카드의 즉시효과도 발동 잊지 말 것
- 팬 획득 효과: general supply에서 대진표 옆으로
- Exhaust pile 효과: face-up으로 쌓아둠 (덱과 섞이지 않게)

## 로봇 (Robot) — 홀수/솔로 상대

### 셋업
- 로봇 대진표(적정 인원용) 배치
- **로봇 시작 카드(S)** = 로봇 덱 (레벨 1 기준)
- 덱 페이즈에서 로봇은 **아무것도 안 함**
- 매치 페이즈 시작 시 덱 셔플 → 매치 룰 그대로 진행 (로봇이 어택 시 카드 리빌)
- 로봇이 이기면 **그 라운드 트로피는 박스로** (플레이어들 못 가져감)
- 매치 종료 후: 로봇 덱과 대진표를 다음 파크 반쪽으로 이동

### 난이도 (레벨 1~5)
| 레벨 | 로봇 덱 조정 |
|-----|-----|
| 1 | S 카드만 |
| 2 | Alpha 카드를 랜덤 R 카드로 교체 |
| 3 | + Beta 카드도 교체 |
| 4 (솔로) | + Good Bot 카드도 교체 |
| 5 (솔로) | + C.H.A.M.P 카드를 2장의 랜덤 R로 교체 |

솔로 룰: SOLO 카드를 다른 R 카드에 섞어서 랜덤 1~5장 추가 시 "special challenge"

---

## 최종전 (Final)

- 7라운드 끝나면 각자 팬 총합 (팬 토큰 + 트로피 뒷면)
- 팬 1위 + 2위가 결승
- **결승 전에는 카드 픽 없음, 제거만 가능**
- 선공은 평소대로 (트로피 번호 큰 사람)
- 결승 승자 = 챔피언
- 동점 시 tiebreaker:
  1. 획득 트로피 수
  2. 여전히 동점이면, 가장 높은 라운드 트로피 소지자

## 시퀀스 다이어그램 요약

```
[게임 시작]
  ↓
라운드 1
  ├─ [덱 페이즈] 자리 이동 → 5장 드로우 → 2장/2장/1장 픽 (레벨 A/B/C에 따라)
  │                → 리롤 (선택) → 덱 카드 제거 (선택)
  └─ [매치 페이즈] 셔플 → 선공 결정 → 어택/방어 교대
                    → 승자 트로피 획득 → 카드 회수
  ↓
라운드 2~7 반복 (매 라운드 다른 상대, 다른 레벨 파일 열림)
  ↓
[결승] 팬 1위 + 2위. 카드 제거만 가능. 승부.
  ↓
[챔피언]
```

## 원작 대비 우리 프로젝트가 신경 써야 할 포인트

1. **카드 순서 랜덤**: 매치 페이즈에서 순서 재배열/조회 금지 → 셔플·드로우 로직 정확히
2. **벤치 6석 종류 제한**: 7번째 다른 이름 카드 발생 시 즉시 패배 감지
3. **파워 계산 순서**: base + bonus 스택, 여러 카드 합산
4. **효과 발동 시점**: 각 키워드별로 확실히 분기 (특히 flag loss → flag possession 순서)
5. **트로피 팬 수**: 뒷 라운드일수록 큰 값. 이길수록 뒤에서 더 얻는 구조
6. **덱 회수**: 매 매치 후 벤치/파크/exhaust 모두 회수 → 다음 라운드 셔플

솔로 이식 시엔 상대 = 로봇으로 고정 → 원작 로봇 룰(레벨 1~5)이 그대로 참고자료
