# 03 · HYPER ROLL ARCADE — 리듬 탭 커밋

## 컨셉 (한 줄)
> **오토배틀 챌린저스**를 TFT 하이퍼롤 템포로 압축한 **60~90초 아케이드 결투** — 자동 리빌 도중 스윕하는 인디케이터를 스윗스팟에서 탭해 "커밋"하는 리듬 스킬 레이어를 얹었다.

## 톤 & 무드

- **아트**: 레트로 아케이드 × 모던 로우폴리 어티튜드 (참고: PEAK, Ball x Pit, Balatro).
  - 카드 = 아케이드 토큰/코인, 픽셀 심볼
  - 벤치 = 트로피 선반 위 미니 캐비닛
  - 깃발 파일 = 콤보 스트릭 타워 (블록 쌓기)
- **사운드**: 8-bit 칩튠 + 모던 서브베이스. 탭 성공 시 슬롯머신 코인 캐스케이드. PERFECT는 두터운 신스 히트 + 픽셀 크런치.
- **타이포**: `Press Start 2P` (히어로/버튼), `VT323` (숫자/카운터), `Space Mono` (마이크로카피).
- **컬러 팔레트**
  | 역할 | 컬러 | HEX |
  |---|---|---|
  | 배경 (딥 네이비) | Cabinet Navy | `#0a0e2a` |
  | 핫 액센트 | Neon Orange | `#ff5e00` |
  | 하이퍼 액센트 | Cyber Magenta | `#ff2bd6` |
  | 정보 액센트 | Electric Blue | `#00c8ff` |
  | 성공/피드백 | CRT Green | `#1aff88` |
  | 서브 그레이 | Screen Ash | `#1a1f3d` |
- **연출 룰**: 픽셀 퍼펙트 모서리 (border-radius 0~2px), CRT 스캔라인 오버레이, 은은한 CRT 곡률(vignette + barrel), 클릭 시 하드 스냅(no easing).

## 코어 루프 (60~90초 매치)

**0~3초 — INSERT COIN**
- "MATCH ▶ ROUND 3" 픽셀 롤인. BGM 인트로.
- 상대 캐비닛(=상대 덱 아트) 슬라이드 인. VS 플래시.

**3~55초 — AUTO REVEAL LOOP (한 카드 = 약 1.5~2.5초 사이클)**
1. 공격자 덱에서 카드 팝업 (0.3s). 캐릭터 픽셀 초상 + POWER 수치.
2. **타이밍 바 등장** (하단 1/4 영역). 좌→우로 인디케이터 스윕 (1.2s).
   - **스윗스팟 존**: 3단계 — 초록 (PERFECT, 폭 8%) / 노랑 (GOOD, 폭 22%) / 빨강 (MISS, 그 외).
   - 탭 안 하면 GOOD로 자동 커밋 (접근성).
3. 탭 결과 → 카드가 콤보 타워에 적층. POWER 합산이 방어자 FLAG 값 도달 시 → **FLAG FLIP!** 스크린 좌우로 갈라지며 깃발이 반대편으로 넘어감.
4. 벤치에 새 유닛 도착 시 캐비닛 픽셀 등장 애니메이션.

**55~90초 — SUDDEN DEATH & RESULT**
- 마지막 라운드 진입 시 스캔라인 붉게, 배경 CRT 왜곡 증가.
- 승패 결정 → 팬 획득 수 카운트업 (숫자 스핀), 리더보드 순위 델타 표시.

## 씬 목록

- **Lobby (아케이드 로비)**
  - 스크린 중앙에 픽셀 캐비닛 캐러셀. 스와이프로 언락된 덱 캐비닛 선택.
  - 상단 CRT 스코어 티커: 오늘 하이스코어, 프리스티지 티어.
  - 하단 큰 CTA: `INSERT COIN → START`.
- **DeckPhase (드래프트)**
  - 하이퍼롤 감성: 3~4장 후보 카드가 슬롯 릴처럼 회전 후 스탑.
  - 리롤 버튼 (제한된 코인), 탭으로 픽. 총 30초 타임어택.
  - 상단에 라운드 프리뷰 (누구랑 붙는지, 상대 덱 캐비닛 실루엣).
- **Match (본편, 이 목업 대상)**
  - 상단: 상대 캐비닛 + 라운드 인디케이터.
  - 중앙: 콤보 타워(플래그 파일) + POWER vs TARGET 카운터.
  - 하단: 현재 리빌 카드 + 타이밍 바 + `TAP!` 버튼.
  - 우측 슬림 컬럼: 벤치 6칸 캐비닛.
- **Result (스코어 리엔트리)**
  - 팬 획득 수가 아케이드 스코어처럼 롤링.
  - 리더보드 이니셜 3글자 입력 (레트로 감성).
  - Next Run 프롬프트 + 프리스티지 상승 여부.

## 메타 레이어

- **아케이드 캐비닛 언락**: 각 캐비닛 = 새 덱 변주(카드 조합 + 아트 팔레트). 스코어 누적 or 도전 과제로 해금.
- **하이스코어 리더보드**: 로컬 저장 우선, 후에 클라우드. 이니셜 3글자.
- **프리스티지 스테이크 (Balatro 오마주)**: RED → GREEN → BLACK → GOLD 스테이크. 상위 스테이크는 상대 파워 +10%, 벤치 자리 -1 등 변수 모디파이어.
- **일일 챌린지 캐비닛**: 고정 시드 덱으로 리더보드 경쟁.

## 시그니처 모멘트

1. **PERFECT TAP** — 화면 100ms 프리즈 → 픽셀 폭발 파티클 → 두툼한 `PERFECT!` 픽셀 텍스트가 CRT 곡면을 뚫고 튀어나옴 → 카메라 살짝 줌인 → 칩 캐스케이드(코인 쏟아짐 SFX). 뇌 도파민 직격.
2. **FLAG FLIP CASCADE** — 콤보 타워가 일제히 반대편으로 도미노 넘어감. 스캔라인이 좌→우로 갈라지는 트랜지션.
3. **SUDDEN DEATH CRT WARP** — 마지막 라운드 배경이 CRT 노이즈 + 배럴 왜곡 심화, BGM 반음 상승. "긴장"이 물리적으로 느껴지는 순간.

## Tradeoffs

- **스킬 감 (+++)**: 리듬 탭이 순수 운을 스킬로 변환. 리텐션·리플레이 강력.
- **아케이드 정체성 (+++)**: 팬층 명확, 마케팅 훅 명확 ("Balatro meets Challengers").
- **접근성 (-)**: 반응속도 낮은 유저에게 불리 → GOOD 자동 커밋으로 완화 필요. 접근성 옵션 필수.
- **원작 충실도 (-)**: 원작의 "지긋한 관전 재미"보다 "짧고 자극적"으로 방향 이동. 원작 팬 반응 갈릴 수 있음.
- **아트 리소스 (--)**: 픽셀 아트 · 파티클 · 사운드 이펙트 = 리소스 부담 큼. MVP는 톤만 잡고 최소로.
- **세션 길이 (+)**: 5~10분 → 60~90초로 압축, 모바일 짜투리 시간에 최적.

## MVP 범위 (2주 빌드)

**Week 1 — 코어 루프 플레이어블**
- 씬 3개: Lobby → DeckPhase(간이) → Match → Result
- 타이밍 바 시스템 (PERFECT/GOOD/MISS + 자동 커밋)
- 콤보 타워 시각화, 플래그 플립 애니메이션
- 카드 5~8종, 캐릭터 4명, 단일 캐비닛(덱) 1개
- PERFECT TAP 시그니처 연출 (프리즈+폭발+SFX)
- 팬 스코어 집계, 로컬 하이스코어

**Week 2 — 아케이드 감성 마감**
- CRT 스캔라인/곡률 셰이더 (CSS filter 스택), 픽셀 폰트 적용
- 사운드 팩: 탭 성공/미스/PERFECT/플립/승패
- 이벤트 로그 UI (COMBO ×N, CHIP RAIN 등)
- 프리스티지 스테이크 1~2단계 (RED/GREEN)
- 리더보드 이니셜 입력 화면
- 접근성 옵션: 스윗스팟 크기, 자동 커밋 강도

**MVP 제외 (v0.2 이후)**
- 캐비닛 언락 다수 (초기엔 1~2개)
- 클라우드 리더보드
- 일일 챌린지 시드
- 커스텀 캐릭터/카드 리네이밍
