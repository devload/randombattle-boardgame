# 제안 #01 · BALATRO CEREMONY

> 네온 카지노 지하 바에서 열리는 심야 칩 세리모니. 모든 리빌은 스코어링 이벤트다.

## 컨셉 (한 줄)

**칩이 쌓이고 무너지는 언더그라운드 카지노 쇼** — 매 카드 리빌마다 숫자가 튀고, 콤보가 붙고, 파일이 넘어갈 때 칩 타워가 우르르 무너지는 Balatro-감성 세리모니.

## 톤 & 무드

- **아트**: 어두운 펠트 테이블 클로즈업. 카드는 물리 포커칩을 얇게 저민 "칩 카드". 숫자는 스탬프 각인처럼 두꺼운 세리프-슬랩. 배경엔 흐릿한 스모크와 무빙 홀로그램 광고판.
- **사운드**: 클링(칩 부딪힘), 츠악(스탬프 각인), 룰렛 트릭틱, 콤보 시 리버브 걸린 신스 스탭. 파일 붕괴 시 챠아앙-와르르 스캐터.
- **컬러 팔레트**:
  - `#0B0F1E` 미드나잇 (배경)
  - `#0F3B2E` 카지노 펠트 그린 (테이블) ← **시그니처**
  - `#FF2E9A` 마젠타 네온 (강조 · 콤보)
  - `#22E9FF` 사이언 네온 (파워 넘버)
  - `#FFD84D` 골드 칩 (레어 · 잭팟)
  - `#E9E4D0` 아이보리 (텍스트)
  - `#7A0E5C` 딥 퍼플 (그림자·비네트)
- **타이포**: 큰 숫자 `Bebas Neue`, 마이크로 텍스트 `Space Mono`, 카드 이름 `Bebas Neue` 자간 넓게.

## 코어 루프 (매치 60~90초)

1. **0s — 오프닝**: "DEALER SHUFFLES" 자막. 양쪽 덱이 툭 놓임.
2. **0~10s — 첫 리빌**: 어택카드가 슬라이드 인 → 숫자가 쿵! 스탬프. 칩 하나 툭 쌓임.
3. **10~40s — 리빌 러시**: 리빌마다 파워 카운터가 튀어오르고, 연속 성공 시 상단에 `COMBO ×2 · ×3` 배너가 강렬하게 팝. 배경 홀로그램이 콤보 배수에 맞춰 밝아짐.
4. **40~55s — 브레이크 임박**: 파일이 4~5장 쌓이면 칩 타워가 위태롭게 흔들. 사이언 광채가 카드 테두리에 물결침.
5. **55~70s — PILE TOPPLES**: 러닝 파워가 방어 파워를 넘어서는 순간 → **와르르**. 칩들이 물리 스캐터로 테이블 밖으로 떨어짐. 화면 셰이크 + `×MULT` 골드 숫자 팝.
6. **70~90s — 정산**: 획득한 팬 수가 계산기 슬롯머신 스타일로 카운트업. 벤치에 새 칩 카드가 딸깍 안착.

## 씬 목록

### LOBBY — "THE VELVET DOOR"
- 지하 바 입구. 네온 간판 `CHALLENGERS 챌린저스` 깜빡. 벨벳 커튼 뒤로 흐릿한 실루엣.
- 메인 CTA는 커다란 골드 칩: "PUSH IN". 서브 CTA는 카드 뒷면 스타일 (STATS / RULES / JOKERS).
- 좌하단에 이번 러닝의 언락된 조커 개수 배지.

### DECK PHASE — "THE DEAL"
- 딜러 시점 오버헤드. 칩카드 5장이 부채꼴로 펼쳐짐 (테이블 위 스포트라이트).
- 카드 탭 → 카드가 손 위로 떠오르며 90도 회전, 뒷면의 효과 텍스트가 스탬프 애니메이션.
- 리롤은 "RESHUFFLE" 룰렛 버튼. 덱 제거 페이즈는 카드를 "BURN" 슬롯에 툭 던지는 인터랙션.

### MATCH — "THE PIT" *(시그니처 씬)*
- 카지노 펠트 배경, 상하 대칭 레이아웃. 상단 상대(딜러 봇), 하단 플레이어.
- 중앙에 **칩 파일 타워** (쌓인 칩카드 3D 스택). 파일 옆에 큰 사이언 파워 넘버.
- 리빌마다 카드가 슬라이드 인 + 파워 스탬프 + 콤보 배너. 벤치는 좌우 6칸 칩 홀더.

### RESULT — "THE CASHOUT"
- 카운터 뒤에 서있는 딜러 실루엣. 팬(스코어)이 골드칩 더미로 시각화.
- 승자에게 골드 트로피 칩 획득 애니메이션. 러닝 히스토리는 영수증(rolled receipt) 스타일 스크롤.
- CTA: "PLAY AGAIN" / "SPEND JOKERS" (메타 언락 진입).

## 메타 레이어

- **JOKER CARDS**: 매 러닝 종료 시 팬 총량에 따라 조커 1~3장 언락. 조커는 **다음 러닝의 덱 룰을 비틈** — 예: "레벨 A 카드 파워 +1", "벤치 첫 슬롯 파워 2배", "콤보 ×2부터 시작". Balatro처럼 최대 5장 장착.
- **VIP RANK**: 우승 누적 시 게스트 → 하이롤러 → 화이트글러브 계급 상승. 계급별로 로비 배경 변화 + 조커 슬롯 확장.
- **NIGHTLY MODIFIER**: 매일 하나의 무료 모디파이어 자동 적용 (예: "오늘 밤엔 모든 파일 시작 파워 +3"). 리롤 불가, 데일리 챌린지 성격.

## 시그니처 모멘트

1. **PILE TOPPLES · 파일 붕괴**
   러닝 파워가 방어를 넘는 그 프레임 — 칩 타워가 물리 시뮬 없이도 CSS 트랜스폼 + framer-motion 스태거로 우르르 스캐터. 화면 살짝 셰이크, 골드 `×MULT` 숫자가 화면 반을 덮음. 소리: 리버브 걸린 챠아앙 + 저역대 붐.

2. **COMBO × CHIP RAIN**
   3연속 리빌 성공 시 상단에 `COMBO ×3` 마젠타 배너, 배경 위에서 작은 네온 칩들이 화면 뒤로 유성처럼 지나감. 파워 카운터가 두 배속으로 카운트업. 폰트가 살짝 흔들리는 저더(judder) 이펙트.

3. **JOKER PROC**
   조커가 발동되는 순간 카드가 잠깐 하얗게 플래시하며 카드 이름 슬랩 스탬프 → 조커의 효과가 텍스트 팝업으로 스탬프 각인. "FREE CHIPS · +5" 같은 문구가 골드로 튀어나옴.

## 현 버전 대비 Tradeoffs

**얻는 것**
- 매 리빌이 이벤트가 됨 → 관전 모드가 지루하지 않음.
- Balatro 감성 = 강한 리텐션 훅. 메타 레이어(조커)로 재플레이 동기 확실.
- 톤이 명확 → 아트/사운드 방향성 결정 쉬움.

**잃는 것**
- 원작의 "카드 게임" 느낌보다 "슬롯머신/카지노" 감성이 강해서 원작 팬 위화감 가능.
- 콤보/조커 시스템은 밸런싱 부담 (숫자 튜닝 리소스 대량 필요).
- 사운드 없으면 매력의 50% 사라짐 → MVP에서 사운드 미룰 수 없음 (MVP 스코프 밖으로 정의된 것과 충돌).
- 시각적 노이즈 많음 → 접근성(모션 감도) 옵션 필수.

## MVP 범위 (2주 빌드 가정)

**반드시 넣을 것**
- MATCH 씬의 칩 카드 슬라이드 인 + 스탬프 파워 넘버 애니메이션.
- 칩 파일 3D 스택 시각화 (CSS transform 계단식).
- 콤보 배너 (×2, ×3만, ×4+ 는 나중).
- PILE TOPPLES 스캐터 애니메이션 (framer-motion stagger + rotate).
- 최소 사운드 3종: 스탬프, 콤보 스탭, 스캐터 (없으면 감성 죽음).
- 컬러 팔레트 + 폰트 확정 → tokens.css로.
- 조커 1장만 하드코딩 (레벨 A 파워 +1) — 메타 티저용.

**미룰 것 (V2)**
- 조커 언락 시스템 전체 (일단 하드코딩 1장).
- VIP RANK 계급.
- LOBBY의 벨벳 커튼 + 네온 간판 시네마틱.
- RESULT의 영수증 롤 애니메이션.
- 데일리 모디파이어.
- 홀로그램 광고판 배경 애니메이션.
- 완성도 높은 조커 아트 (텍스트 프레임만 MVP).
