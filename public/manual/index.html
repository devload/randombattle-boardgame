// 30 SECOND GUIDE

Quick Start

  1. NEW TOURNAMENT 눌러 시작
  2. 매 라운드 카드 2~3장 픽
  3. 매치는 자동 진행 — 관전만 하면 됨
  4. 7라운드 끝나고 팬(Fans) 1등이면 챔피언

아래는 화면별로 뭘 봐야 하는지 그림으로 정리한 가이드입니다.

// 01

Lobby · 시작 화면

NEW TOURNAMENT 버튼 하나 누르면 시작.

우측 하단 dev 버튼은 개발용 씬 전환기 — 무시하세요.

// 02

Tournament Board · 매 라운드 상황판

여기서 확인하는 것

  • ROUND 04 / 07 — 지금 몇 라운드인지
  • LEADERBOARD — 팬 순위. YOU 가 사이언으로 강조
  • NEXT MATCH — 이번 라운드 상대 봇 + 성격 힌트
  • DRAFT PILES OPEN — 뽑을 수 있는 카드 파일 (A / B / C)

ENTER THE ARENA 누르면 덱 페이즈로.

// 03

Deck Phase · 카드 픽

어떻게 픽하나

  • 5장이 뜬 카드 중 2장 탭 → 옐로 외곽선 = 선택됨
  • 다음 파일(B)에서 2장 더. R5+ 는 B 두 장 vs C 한 장
  • 마음에 안 들면 REROLL (파일마다 1회)
  • 다 뽑으면 CONFIRM PICKS → 매치

파워 상한 — A ≤ 3 · B ≤ 5 · C ≤ 10

// 04

Match Phase · 자동 대전

화면 읽는 법

  • 위 = 상대, 아래 = 나
  • 중앙 🚩 카드 — 지금 깃발을 가진 카드
  • 양쪽 6칸 = 벤치. 7종째 채우는 사람이 패배
  • 파워 MY POWER ≥ FLAG 되면 깃발 뺏김
  • 매치는 자동. TAP 모드로 한 이벤트씩 넘길 수도 있음

승리 조건 (둘 중 하나)

  • 상대가 덱을 다 소진하고도 못 뺏음
  • 상대 벤치가 7종 오버플로
// 05

Result · 최종 결과

7라운드 후 팬(Fans) 1등 = 챔피언.

팬은 트로피 + flag-loss 카드가 벌어준 팬의 합산.

PLAY AGAIN으로 다시 도전.
