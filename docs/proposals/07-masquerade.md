# 07 · MASQUERADE · 가면의 무도회

> **오리지널 컨셉 (실존 게임 아님)**
> 세 프로젝트(Sagrada · RandomBattle · Faraway)의 DNA를 조합한 신작 아이디어.

## 한 줄 컨셉

**하나의 왕궁, 8개의 자리, 두 사람이 다투는 무도회.** 얼굴 없는 손님을 자리에 놓고, 5라운드에 정체가 드러난다.

## 🎭 어떤 느낌인가 (목업 3장)

### 전반부 · 왕궁 (R3/8, 가면 착용 중, 자리 다툼 시작)
![gallery](./images/07-masquerade/gallery.png)

**공유 왕궁 8자리** · 왼쪽 하석부터 오른쪽 왕 옆까지. 이미 4자리 채워짐: Slot 1,3은 내 손님 (골드 테두리), Slot 2,4는 상대 손님 (실버 테두리), 모두 마스크 착용. 우측 아래 **추방석**엔 지난 결투 패자 카드 1장. R5 UNMASKING 예고 배너.

### R5 · GRAND UNMASKING (시그니처 컷씬)
![unmasking](./images/07-masquerade/unmasking.png)

화면 어두워지고 이미 놓인 자리들 위 스포트라이트. 마스크 회전 벗겨짐, 각자 정체 색상 오라 발현. 후반부부턴 정체 아는 상태로 자리 다툼.

### 채점 세리모니 · RECKONING (게임 종료 후)
![reckoning](./images/07-masquerade/reckoning.png)

왼쪽부터 자리별 채점. 각 자리 소유자 확인 → 정체 규칙 발동 → 점수 반영. 우측엔 최종 추방 페널티 정산. 상단 라이브 스코어보드.

## 왜 이 게임인가

세 프로젝트의 약점을 각 게임의 강점으로 커버:

| 이전 게임 | 약점 | Masquerade 해결 |
|---|---|---|
| Sagrada | 결정 밀도 낮음 | 매 라운드 자리 선택 + 상대 예측 |
| RandomBattle | 매치 중 개입 0 | **매 라운드 상대와 자리 다툼** |
| Faraway | 병렬 solo · 중반 드라마 없음 | **공유 왕궁 + R5 UNMASKING 폭탄** |

**핵심 혁신**: 각자 자기 갤러리 만드는 게 아니라 **하나의 왕궁 8자리를 두 사람이 다투는** 구조.

## 코어 시스템 · 공유 왕궁 ⭐

### 왕궁 구조

**하나의 갤러리, 8자리** (Slot 1 하석 ~ Slot 8 왕 옆 상석)

```
[Slot 1] [Slot 2] [Slot 3] [Slot 4] [Slot 5] [Slot 6] [Slot 7] [Slot 8]
  하석                                                            상석
```

- 두 플레이어 다 이 왕궁에 카드 놓음
- 각 자리 = 최대 1장 (소유권 다툼)
- 자리마다 나만의 소유 or 상대 소유

### 준비
- **손패**: 3장 (모두 마스크 착용)
- **왕궁**: 8자리 · 비어있음
- **시장**: 마스크 카드 3장 face-up
- **추방석**: 두 사람 각자 하나씩 (진 카드가 여기로)

### 손님 카드

**표면 정보** (항상 보임)
- 이름 (예: "붉은 나비 부인")
- 표면 점수 (0-4)
- 카드 번호 (1-30, 학자 규칙용)
- 표면 아이콘 (사냥꾼 규칙용)

**숨겨진 정체** (R5까지 비공개)
- 5종 중 하나: 왕족 · 도둑 · 학자 · 사냥꾼 · 신비주의자

### 정체 카테고리 (5종 · RPS-5)

| 정체 | 색 | 채점 규칙 (자기 자리에서만 발동) | 이김 | 짐 |
|---|---|---|---|---|
| **👑 왕족** | 골드 `#d4a574` | 왕궁 내 왕족 자리 1개당 +3 (양쪽 다 카운트) | 도둑 · 신비주의자 | 학자 · 사냥꾼 |
| **🗡 도둑** | 다크 레드 `#8b0000` | 인접 자리가 상대 카드이면 상대 표면 점수의 절반 훔침 | 학자 · 사냥꾼 | 왕족 · 신비주의자 |
| **📚 학자** | 짙은 청 `#4a5c6a` | 자리 번호(1-8) = 카드 번호(1-30 mod 10) 일치 시 +8 | 왕족 · 신비주의자 | 도둑 · 사냥꾼 |
| **🏹 사냥꾼** | 이끼 그린 `#88a065` | 자기 표면 아이콘 + 인접 카드 아이콘 겹치면 +5 | 왕족 · 학자 | 도둑 · 신비주의자 |

**RPS-5 서사**:
- 왕족 crushes 도둑 (권위로 처벌), 신비주의자 (교회 탄압)
- 도둑 outsmarts 학자 (지식 훔침), 사냥꾼 (도주)
- 학자 debunks 왕족 (역사서 기록), 신비주의자 (미신 반박)
- 사냥꾼 hunts 왕족 (사냥), 학자 (침묵)
- 신비주의자 curses 도둑 (저주), 사냥꾼 (환상)

## 라운드 흐름 · 매 라운드 자리 다툼

### 표준 라운드 (R1 ~ R4 · 전반부, R5 UNMASKING 이후 R5 ~ R8 · 후반부)

**Step 1 · 카드 & 자리 커밋 (동시, face-down)**
- 손패 3장 중 1장 선택
- 왕궁 8자리 중 하나 선택 (비어있는 자리만)
- face-down으로 커밋

**Step 2 · 동시 리빌**
- 두 사람 카드+자리 동시 공개

**Step 3 · 판정**

**Case A · 서로 다른 자리 선택**
- 두 카드 각각 그 자리에 배치 (마스크 착용 상태로)
- 결투 없음
- "평화" — 우아한 무도회

**Case B · 같은 자리 선택** → ⚔️ **결투 발동**
- 전반부 (R1-R4, 정체 비공개): 결투 결과 **UNMASKING까지 유예** — 두 카드 모두 그 자리에 face-down 스택
- 후반부 (R6-R8, 정체 공개): **즉시** RPS-5 결투
  - 승자 카드가 자리 차지
  - 패자 카드는 **추방석**으로 (-3 페널티)
- 무승부 (같은 정체): "동맹" — 자리에 두 장 다 남아있음 (양쪽 표면 점수 획득, 정체 보너스는 반반)

**Step 4 · 드로우 (Faraway 스타일 순차 다이얼로그)**
- 성소 카드 (오름차순 유지 시)
- 지역 카드 필수 뽑기

**Step 5 · 다음 라운드**

### R5 · GRAND UNMASKING ⭐⭐ (시그니처 컷씬)

R5 배치 완료 후 자동 트리거:

**시퀀스** (약 10초):

1. **정체 리빌** (4초):
   - 화면 어두워짐
   - 왕궁 자리들 위 스포트라이트 점등
   - 각 자리 카드 마스크 물리 회전 벗겨짐
   - 정체 아이콘 페이드-인, 오라 발현

2. **유예된 결투 정산** (4초):
   - R1-R4에 face-down 스택된 결투들이 이제 판정됨
   - 슬롯별 순차 결투 시각화
   - 승자 카드 자리 차지, 패자 카드 추방석으로
   - 극적 순간: 왕궁 지형 재편성

3. **후반부 전환 알림** (2초):
   - "이제 정체는 알려졌다 · 후반부 시작"
   - 시장 카드도 정체 공개 상태로 전환

### 후반부 라운드 (R6 ~ R8)

- 표준 라운드와 같지만 **결투 즉시 판정**
- 손패·시장 카드 다 정체 공개
- 서로의 정체 조합 다 보이는 상태에서 자리 다툼

## 🏆 RECKONING · 채점 세리모니

R8 종료 후 자동 시작. 왼→오 스캔:

**각 자리 처리** (1.5초씩):
1. 자리 스포트라이트, 소유자 표시 (내 것 골드 / 상대 것 실버)
2. 표면 점수 획득 (소유자에게)
3. 정체 규칙 발동
4. 정체 상호작용 (도둑 훔침 등)
5. 총점 실시간 반영

**추방석 정산** (모든 자리 스캔 후):
- 각자 추방 카드 개수 × -3 페널티 차감
- 극적 시각화: 추방된 카드 하나씩 붉게 fade

**콤보 판정**:
- 🏰 **왕궁 지배**: 6자리 이상 소유 → +15
- 🎭 **완벽한 무도회**: 5정체 다 등장 (양쪽 카드 합) → +10 (양쪽)
- 👑 **가문의 위엄**: 한 정체 4명 이상 소유 → +15
- 🕊 **평화의 무도회**: 결투 없이 8라운드 (다 다른 자리) → +20 (양쪽) — 매우 드묾

**최종 판정**: 나 총점 vs 상대 총점

## 심리전 축 · 매 라운드 결정 3가지

### 1. 어느 자리에 놓을까?

**자리마다 정체별 유리도 다름**:
- **Slot 1-3** (하석) : 왕족 스택 시작 위치 · 학자 매치 가능 (카드 번호 1-30 중 낮은 것)
- **Slot 4-5** (중석) : 사냥꾼 인접 시너지 · 학자 매치 확률 최고
- **Slot 6-8** (상석) : 신비주의자 완성 · 왕족 왕 옆 자리

**전략**:
- 자기 카드가 왕족이면 Slot 1-3 시작 후 계속 채워 스택
- 학자면 슬롯 번호 == 카드 번호 매치 자리
- 사냥꾼이면 상대 카드 인접 자리 노려서 아이콘 겹칠 자리

### 2. 상대 회피 vs 정면 승부?

**회피 (다른 자리)**
- 확실한 획득 · 결투 없음
- 하지만 좋은 자리는 놓치기 쉬움

**정면 승부 (같은 자리)**
- 이기면 그 자리 확보 + 추방 페널티 안 받음
- 지면 카드 추방 (-3)
- 상대 정체 정확히 예측해야 함

**심리 예측 방법**:
- 전반부: 상대 표면 점수·이름으로 정체 추리
  - 표면 점수 낮음 → 정체 보너스 큰 정체 (신비주의자 or 학자) 가능성 up
  - "부인·자작" 등 귀족 이름 → 왕족 확률 up
  - 카드 번호 낮음 → 학자 자기 매치 슬롯 노릴 가능성
- 후반부: 이미 공개된 정체 조합으로 남은 손패 추리
  - 상대 손패에 도둑 남았는지, 왕족 남았는지 알기

### 3. 손패 관리 · 언제 뽑을 카드가 무엇일까?

- 시장에서 뽑는 카드 = 미래 무기
- 표면 점수 낮은 카드 = 정체 보너스 클 카드 (도박)
- 카드 번호 = 학자 매치용
- 아이콘 = 사냥꾼용

## 예시 시나리오 (한 판 상상)

**R1**: 나 붉은 나비 부인 (표면 +3) → Slot 3에 face-down. 상대 가스통 자작 (표면 +4) → Slot 5. 서로 다른 자리, 배치 완료.

**R2**: 나 은빛 여우 남작 (표면 +2) → Slot 3? 상대도 Slot 3? 이미 R1에 내 카드가 있음. Slot 3 만석이니 다른 자리 선택 강제.
- 나 → Slot 2 선택
- 상대 → Slot 6 선택

**R3**: 나 미스터리 손님 (표면 +2) → Slot 5 (상대 옆에 도둑 예상). 상대도 Slot 5 → **같은 자리! 전반부라 결투 유예, face-down 스택**

**R4**: 손패 재정비, 다른 슬롯 배치

**R5 UNMASKING** ⭐:
- 6장 정체 다 공개
- 유예된 Slot 5 결투 정산: 내 카드 도둑 vs 상대 카드 사냥꾼 → 사냥꾼이 도둑에게 짐. 내 도둑 승리. 상대 카드 **추방석**으로.
- 극적 순간: 상대 -3, 내 카드 자리 확보

**R6-R8**: 정체 아는 상태에서 자리 다툼. 결투 즉시 판정.

**RECKONING**: 왼→오 스캔. 콤보 판정. 승자 결정.

- 두 카드가 화면 양쪽으로 이동 · 정체 아이콘 표시
- RPS-5 매치 애니메이션 (예: 왕족이 도둑을 짓누름)
- 승자 자리 확보 골드 팝, 패자 추방 붉은 팝

### 4. GRAND UNMASKING (컷씬)
- 화면 어두워짐 · 스포트라이트
- 마스크 벗기 순차 애니메이션
- 유예된 결투들 순차 정산
- 후반부 진입 알림

### 5. RECKONING (채점 세리모니)
- 왼→오 자리별 스캔
- 정체 상호작용 시각화
- 추방 페널티 정산
- 콤보 판정 팝업

### 6. RESULT
- 최종 스코어
- 승리/패배 판정
- 이번 무도회 요약 (결투 승률, 추방 수 등)
- 다음 무도회 CTA

## 시그니처 모먼트

### 1. THE FACE-DOWN COMMIT ⭐ (매 라운드)
- 두 카드가 face-down으로 자리에 놓임
- 서로 뭘 놓았는지 모름 · 심장 뛰는 순간
- 리빌 → 같은 자리? 다른 자리? 결투?

### 2. GRAND UNMASKING (R5)
- 6장 정체 동시 공개
- 유예된 결투들 정산
- 판이 근본적으로 바뀌는 순간

### 3. DUEL RESOLUTION (후반부 매 라운드)
- 상대의 정체 아는 상태에서 정면 승부
- 상성 정확히 예측했다면 이김
- 상대의 노림수를 카운터로 되받음

### 4. EXILE (누적)
- 결투 진 카드가 하나씩 추방석 쌓임
- 4장 → 위험, 5장 → 즉시 패배
- 시각적으로 추방석이 화려한 붉은 오라 발산

### 5. WANG-GUNG DOMINATION (콤보)
- 8자리 중 6+ 소유 → "왕궁 지배자" 배너
- 상대의 자리 하나씩 뺏어가는 마지막 스퍼트

## 현 프로젝트(Faraway) 대비 Tradeoffs

### 얻는 것
- ✅ **진짜 결투 게임** — 매 라운드 상대와 상호작용
- ✅ **심리전** — 자리 선택 + 정체 예측 + 블러프
- ✅ **극적 리빌** — R5 UNMASKING 순간
- ✅ **누적 페널티** — 추방석 = Challengers 오버플로 감성
- ✅ **콤보 상호작용** — 정체 5종 조합
- ✅ **감성 완전 다름** — 왕궁 정치 무도회

### 잃는 것
- ❌ **미검증** — 새 시스템 재미 확신 불가
- ❌ **복잡도** — 룰 5-10분 학습 필요 (Faraway는 3-5분)
- ❌ **처음부터** — 카드 40장 + 정체 5종 완전 밸런싱
- ❌ **아트 리소스** — 가면 초상화 40장, 왕궁 배경, 결투 애니메이션
- ❌ **UI 복잡** — 공유 왕궁 + face-down 커밋 UX 신설

## 결정 필요 사항

### A. 자리 배치 UI · Face-down or Face-up?
- **Face-down (커밋)**: 심리전 최대 · UI 복잡 (자리 선택 + 카드 선택 별도)
- **Face-up (즉시)**: 단순 · 하지만 심리전 약함
- **하이브리드**: 후반부만 face-up

### B. 전반부 결투 유예 룰
- **완전 유예** (현재): UNMASKING 때 정산 · 극적
- **표면 점수만 비교**: 즉시 결과 · 정체 심리전 약해짐
- **선택 유예**: 나만 유예 결정 가능

### C. 정체 밸런스
- 도둑이 다른 정체 대비 페널티 크지 않은지
- 신비주의자 조건이 너무 유동적인지
- 왕족 스택이 너무 강한지

### D. 손님 이름 스타일
- 유럽 클래식 · 판타지 시적 · 동양적

### E. 조기 종료 임계값
- 5장 추방 = 몰락? 4장? 6장?
