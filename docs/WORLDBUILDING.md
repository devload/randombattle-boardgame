## 세계관 (2페이지 요약)

**Neo-Seoul, 20XX년.** 대기업들이 도시를 나눠 갖는 시대. 하지만 시민들의 눈길을 사로잡는 것은 매주 열리는 **AUTO-DRAFT ARENA** — 스폰서 없이 참가할 수 있는 유일한 상금 리그.

플레이어는 자기 팀을 즉석에서 조합해서 다른 러너들과 붙는다. 이길수록 팬(Fans)이 붙고, 팬이 챔피언을 만든다. 관중은 홀로그램 스탠드에서 함성을 지르고, 배당은 실시간으로 뒤집힌다.

**당신은 이제 막 시그넷을 딴 신인 러너다.** 상대는 셋 다 AI 봇. 봇들은 각자 후원사가 있고, 각자 카드 스타일이 다르다. 7라운드를 뚫으면 챔피언 자리에 앉는다.

## 3개 카드 세트

### 1. BASIC — Downtown (거리 · 뉴트럴)
- 스트리트 러너, 몬 블레이드, 스팀팩, 대몬 프로세스 등
- 파워는 평범하지만 견고. 일부는 flag-loss에서 팬을 되돌려줌
- 컬러: 사이언 위주. 대표 카드 → `DAEMON.EXE`, `APEX GUARD`

### 2. CORP OPS — Zaibatsu (기업)
- 컨셉: 기업의 SEC 팀 + 임원 + 홀로그램 위장
- 메커닉: **벤치 시너지** — 같은 카드가 많을수록 파워가 배수로 뜀
- 대표 카드 → `MEGACORP CEO`, `HOLOGRAM DECOY`, `EXEC.AI`
- 톤: 마젠타, 골드 액센트

### 3. UNDERGROUND — Netscape (해커·게릴라)
- 컨셉: 스크립트 키디, 넷러너, 데이터 파이어러시
- 메커닉: **즉시 파워 + 어택 부스트** — 짧은 매치를 폭발적으로 뒤집기
- 대표 카드 → `KRAKEN.NET`, `GLITCH ARCHON`, `NETRUNNER`
- 톤: 사이언·네온 그린

## 카드 이름 & 저작권 방침

- **모든 카드 이름은 재작성** — 원작 *Challengers!* 의 이름 그대로 쓰지 않음
- **효과 텍스트도 재작성** — 기능은 동일해도 문구는 우리 것
- **아트는 자체 제작** — MVP는 이모지, V2는 커스텀 아이콘/픽셀
- **게임 이름** — "Challengers"는 브랜딩상 회피, 프로젝트명 `RandomBattle` (변경 여지 있음)
- **README에 팬 리메이크 면책 명시**

## 시각 스타일 가이드

### 카드
- 홀로그램 카드 (반투명 + 네온 테두리)
- 레벨별 색: S=회색, A=네온그린, B=사이언, C=마젠타
- 카드 이름은 Orbitron 900 · 파워는 대형 숫자로 강조
- 배경: 다크 그라디언트 + 홀로 스트라이프

### 씬
- 배경: 사이버 그리드 + CRT 스캔라인
- HUD: 육각 클립 버튼, 옐로 액센트
- 애니메이션: 카드는 슬라이드 인, 파워는 카운트업, flag 이동은 회전 스프링

### 오디오 (V2)
- 매치 진행 중 로우파이 신스 루프
- 카드 리빌 시 스위치 클릭 SFX
- 매치 승리 시 아나콜리 신스 스팅어
