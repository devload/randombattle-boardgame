# 05 · CHALLENGERS ROLL

> **"TFT의 상점 밀도 + Balatro의 곱셈 스코어링 + Challengers!의 벤치 오버플로"**
> 매 라운드가 **30초 상점 → 45초 배틀** 두 페이즈로 쪼개진 7라운드 로그라이트 토너먼트.

---

## 한 줄 컨셉

7라운드 각 라운드가 **바쁜 상점 30초** + **곱셈 배틀 45초** 로 쪼개져, 관전만 하는 게 아니라 매 라운드 마이크로 결정 20~30개를 내려야 하는 챌린저스 리메이크.

## 왜 이건 재미있을까 (설계 원리)

TFT와 Balatro가 재미있는 이유는 **준비 페이즈의 밀도** + **결과 페이즈의 감정 이입**. 자동 리빌 자체가 문제가 아니라 "내가 짠 조합이 터지는 걸 보는" 감정 이입이 없는 게 문제였음.

- **바쁨을 만든다**: 30초 안에 골드 관리·리롤·구매·배치를 다 해야 함. TFT식 시간 압박.
- **감정 이입을 만든다**: 라인업을 내가 짜니까 결과가 내 작품. 콤보 배수가 터지면 내 승리감.
- **롱테일 러닝**: 조커 언락·보스 블라인드·캐러셀로 매 러닝 다름.
- **원작 정신 유지**: 벤치 6종 오버플로 · 팬 스코어 · 카드 효과 시스템 다 재사용.

## 세션 흐름

```
러닝 시작
  ↓
[R1] 상점 30s → 배틀 45s     ← 기본 라운드
[R2] 상점 30s → 배틀 45s
[R3] 상점 30s → 배틀 45s → 캐러셀 (무료 카드 1장 픽)
[R4] BOSS 상점 30s → 배틀 45s  ← 특수 페널티
[R5] 상점 30s → 배틀 45s
[R6] 상점 30s → 배틀 45s → 캐러셀
[R7] BOSS 상점 30s → 배틀 45s  ← 마지막 페널티
  ↓
결승: 팬 1·2위 격돌 (상점 45s + 배틀 60s, 조커 全 활성)
  ↓
러닝 결과 → 조커 언락 → 다음 러닝
```

총 세션: **10~13분**. TFT Hyper Roll 카덴스.

## 상점 페이즈 (30초)

### 리소스
- **골드** — 라운드 시작 시 자동 +2. 이월 가능. 이자 없음(TFT 이자는 복잡해서 v1엔 뺌).
- **레벨** — 초기 1. 티어 확률에 영향. 골드 4로 +1 레벨업 가능.
- **시간** — 30초 카운트다운. 만료 시 자동 배틀 진입.

### 상점 슬롯 (5개)
매 라운드 자동 스폰. 카드마다 티어와 가격 표시:

| 티어 | 가격 | 등장 확률 (Lv별) |
|------|------|-----------------|
| T1 · Common    | 1G | Lv1: 100%, Lv3: 40% |
| T2 · Uncommon  | 2G | Lv2: 60%, Lv3: 40% |
| T3 · Rare      | 3G | Lv3: 20%, Lv4: 30% |
| T4 · Epic      | 4G | Lv4: 15%, Lv5: 25% |
| T5 · Legendary | 5G | Lv5: 5% |

### 액션
1. **BUY** — 카드 탭 → 골드 지불 → 라인업으로 이동
2. **REROLL** — 2G로 상점 5장 셔플
3. **LEVEL UP** — 4G로 플레이어 레벨 +1
4. **LINEUP DRAG** — 획득 카드를 라인업 슬롯 6칸에 원하는 순서로 배치. 순서 = 배틀에서 뽑히는 순서
5. **SELL** — 라인업 카드를 매대에 다시 던지면 절반 가격 환급

### 스택 승격 (Balatro 조커 곱셈감)
- 같은 이름 카드 **3장** 소유 시 자동 승격 → 한 단계 상위 능력 카드
- 승격 카드 다시 3장 → 재승격 (2티어 승격)
- 승격 시 시그니처 애니메이션 (3D 회전 + 골드 스파크)

### 다음 상대 프리뷰
상점 화면 우상단에 "다음 상대: DAEMON.EXE" 미니 초상 노출. 상대 특성 태그(예: "ATTACK-HEAVY")로 상점 전략 유도.

## 배틀 페이즈 (45초)

### 라인업 큐 (신규)
상단에 라인업 6장이 세로/가로 스택으로 보임. **다음 뽑힐 카드가 하이라이트**되어 예측 가능. Balatro의 "다음 손패" 감각.

### 오토 리빌
- 내 라인업 순서대로 카드가 뽑히며 아레나로 슬라이드 인
- 상대도 병렬로 뽑음 (기존 게임 로직 재사용)
- 리빌 순간마다 파워 카운트업 + 콤보 계산

### 콤보 시스템 (Balatro 감성)
카드마다 **태그** 1~2개 부여 (예: HACKER · WARRIOR · GHOST · BOOST · INSTANT).

- 연속 리빌된 카드가 태그 공유 → 콤보 배수 스택
- HACKER 3연 = 배수 ×3
- HACKER 5연 = 배수 ×5
- 다른 태그 리빌 시 콤보 끊김 (배수는 리셋)
- 브리치 시 **배수만큼 팬 폭발** — Balatro의 `MULT×N` 감성

### 스킬 탭 (선택적)
리빌 순간 화면 아무데나 탭 → 이 카드에 즉시 +1 파워
- 라운드당 3회 제한
- 강제 아님. 놓쳐도 게임 진행.
- Balatro의 "PLAY THIS HAND!" 감성 유지

### 시간 종료
45초 만료 시 즉시 정산. 그때까지의 팬 합산.

## 로그라이트 축

### 조커 (Balatro 그대로)
러닝 종료 시 팬 총량에 따라 조커 1~3장 언락. 다음 러닝에 최대 3장 장착.

**예시 조커**
- `SHUFFLE MASTER` — 리롤 1G (기본 2G)
- `COMBO ADEPT` — 콤보 기본 배수 +1
- `PRICE FIXER` — 티어 1 카드 무료
- `BENCH GARDENER` — 벤치 스택 시 파워 +1
- `LEGENDARY LURE` — Lv5 도달 시 첫 상점에 T5 카드 보장
- `RECESSION` — 골드 이자 지급 (10G 초과 시 +1)

### 보스 블라인드 (Balatro 그대로)
4·7라운드는 특수 페널티. 상점 진입 전 극적 등장 컷씬.

**예시 보스**
- `PURGE`      — T1 카드만 상점에 등장
- `SILENCE`    — 이 라운드 조커 全 비활성
- `GLASS ECON` — 골드 0으로 시작
- `OVERHEAT`   — 벤치 슬롯 5개로 축소
- `RUSH HOUR`  — 상점 시간 15초로 단축
- `HEAVY TAX`  — 모든 구매 +1G

이기면 **큰 보상**: 골드 +5, 팬 +10, 조커 언락 확률 부스트.

### 캐러셀 (TFT 그대로)
3·6라운드 종료 시 회전 접시 등장. 카드 4장 회전 중.
- 카운트다운 5초
- 다른 봇들이 카드 픽업 애니메이션 (경쟁 압박)
- 마지막에 남은 카드 → 내 라인업 자동 편입

## 씬 목록

### LOBBY (재활용)
현 씬 유지. 러닝 히스토리 배지 + 언락된 조커 개수 노출.

### SHOP PHASE ★ (신규 · 시그니처)
- 상단: 30s 카운트다운 · 골드 · 레벨 · 라운드
- 중단: 상점 슬롯 5칸 (카드 매대 그리드)
- 하단: 라인업 슬롯 6칸 (드래그 존)
- 우측 사이드바: 다음 상대 프리뷰 · 리롤/레벨업 버튼

### BOSS BLIND (신규)
- 상점 진입 직전 컷씬
- 화면 어두워짐 → 보스 실루엣 슬라이드 인 → 페널티 스탬프
- 3초 후 상점으로 자연 전환

### BATTLE PHASE ★ (기존 개선)
- 상단: 라인업 큐 (다음 뽑힐 카드 하이라이트)
- 중앙: 아레나 (진영별 카드 리빌, 콤보 배수 팝)
- 하단: 팬 카운터 · 시간 카운트다운
- 스킬 탭 인디케이터 (3회 남음)

### CAROUSEL (신규)
- 회전 접시 4개 카드
- 5초 카운트다운
- 봇 실루엣이 카드 픽업 애니메이션
- 마지막 카드 픽업 시 라인업 편입 팝업

### RESULT (재활용)
- 러닝 팬 총합 · 승률
- 언락된 조커 카드 오픈 씬
- CTA: "다시 도전" / "조커 룸"

### JOKER LAB (신규)
- 획득 조커 카드 진열
- 다음 러닝 장착 슬롯 3개
- 조커 조합 프리뷰 (총 예상 승률/보너스)

## 시그니처 모멘트

### 1. STACK PROMOTE 스택 승격 ★
같은 이름 3장 채워지는 순간 → 카드 3장이 화면 중앙에 모여 → 3D 회전 + 골드 스파크 폭발 → **승격 카드**로 통합.
- 사운드: 낮은 붐 + 챠임 + 텍스트 스탬프 "PROMOTED"
- 승격 카드에 "★" 뱃지 붙음
- 감정: "내가 만들어냈다"

### 2. COMBO ×N 배수 폭발 ★
콤보가 스택될 때마다 화면 상단에 큰 배수 텍스트가 점점 커짐:
- ×2 녹색 소박함
- ×3 노란 흥분
- ×5 주황 위험
- ×8 마젠타 광기
- 브리치 시 배수 만큼 팬 숫자가 슬롯머신 스타일로 카운트업
- 사운드: 배수마다 피치 상승. 최고 배수 시 리버브 걸린 신스 스탭

### 3. BOSS BLIND ENTRANCE 극적 등장
4·7라운드 시작 시:
- 화면 밝기 30% 다운
- 보스 이름 카드 슬라이드 인 (예: `PURGE · 정화의 일격`)
- 페널티 텍스트 각인 애니메이션 ("T1 카드만 등장")
- 3초 후 상점으로 전환
- 감정: "이번 라운드는 특별하다"

### 4. CAROUSEL RUSH (TFT 감성)
접시가 돌 때 다른 봇들의 실루엣이 접시 주변에 몰림 → 하나씩 픽업 → 카운트다운 압박 → 내가 픽업 → 라인업 편입 팝업.
- 사운드: 회전 중 저역대 앰비언스 → 픽업 시 챠악!

## 현 버전 대비 Tradeoffs

**얻는 것**
- ✅ 액션감 대폭 상승 (30초 상점 = 손이 바쁨)
- ✅ 감정 이입 (내가 짠 라인업이 배틀에서 터짐)
- ✅ Balatro 감성 콤보 배수 = 숫자 뽕
- ✅ 조커 언락 = 러닝 간 다양성
- ✅ TFT/Balatro 팬층에 익숙한 UX 패턴
- ✅ 기존 카드 시스템·벤치 로직 재활용 가능

**잃는 것**
- ❌ 세션 시간 증가 (5-10분 → 10-13분)
- ❌ UI 복잡도 상승 (상점 페이즈 UI 신설)
- ❌ 밸런싱 리소스 증가 (콤보/조커/보스 튜닝)
- ❌ 원작 룰과 거리감 (챌린저스 오리지널 팬 위화감 가능)
- ❌ 학습 곡선 (신규 UX 온보딩 필요)
