# 06 · FARAWAY · SOLO MOBILE

> **"카드는 왼쪽에서 오른쪽으로 놓지만, 점수는 오른쪽에서 왼쪽으로 계산한다"**
> 미래를 예측하며 현재를 배치하는 15분 판타지 타블로 빌더.

원작: **Faraway** — Corentin Lebrat & Johannes Goupy, 2023.
Kennerspiel des Jahres 2024 수상. Nominated Golden Geek Best Card Game.

---

## 컨셉 한 줄

**카드 배치 방향과 점수 계산 방향이 반대인 15분 타블로 빌더** — 이번 턴 놓는 카드가 "미래의 스코어러"가 될지, "미래 스코어러를 위한 셋업"이 될지, 혹은 "성소 접근권을 위한 오름차순 유지"인지 매 라운드 3중 결정.

## 원작 룰 완전 정리

### 카드 종류

**1. 지역 카드 (Region · 68장)**
게임의 메인. 각 카드는 아래 4가지 정보를 담음:

```
┌─────────────┐
│ 34          │  ← 카드 번호 (1~68) · 턴 순서 결정
│             │
│ [일러스트]   │  ← 지역 풍경
│             │
│ ─────────── │
│ 조건: 🌙   │  ← 조건 아이콘 (없거나 1~N개)
│ 점수: +5    │  ← 조건 충족 시 얻는 값
│ 제공: 🌙🌙  │  ← 타블로에 활성화되는 아이콘
└─────────────┘
```

- **번호**: 뽑기 순서와 성소 접근권 결정 (아래 참조)
- **조건**: 스코어링을 위해 필요한 아이콘 (예: `3× 🌙`) · 없는 카드도 있음
- **점수**: 조건 충족 시 즉시 획득 · 실패 시 0점
- **보상 아이콘**: 놓으면 타블로에 즉시 활성화 → **오른쪽에 놓일 미래 카드의 조건 재료**로 쓰임

카드 아이콘 종류 (원작 기준 대략 5-6종):
- 🌙 밤 (Night)
- ☀️ 낮 (Day)
- 🐚 조개 (Shell)
- 🪶 깃털 (Feather)
- 🔮 수정 (Crystal)
- 🌸 꽃 (Flower)

**2. 성소 카드 (Sanctuary · 54장)**
지역보다 강력한 특수 카드. 두 종류:

- **즉시 발동형**: 놓거나 발동할 때 즉시 효과 (예: 특정 지역 카드 아이콘 복제, 조건 무시, 손패 2장 획득)
- **스코어링형**: 게임 종료 시 특수 점수 (예: `🌙 아이콘 × 2점`, `왼쪽 짝수 번호 지역 × 3점`, `가장 큰 지역 번호 × 1점`)

성소는 손패 위쪽에 별도 슬롯으로 보관 · 라운드마다 원하는 시점에 발동 가능.

### 준비

1. 지역 덱 셔플 → 각 플레이어에게 **3장 손패**로 배분
2. 지역 카드 **3장을 앞면**으로 공용 시장에 공개
3. 성소 덱 셔플 → **3장을 앞면**으로 공용 성소 시장에 공개
4. 각 플레이어 앞에 **8칸 타블로** (실제로는 카드 놓을 공간만 정해두는 것, 물리적 보드 없음)

### 8라운드 반복 흐름

각 라운드 5스텝으로 구성:

**Step 1 · 동시 카드 선택 (Simultaneous Select)**
- 모든 플레이어는 손패 3장 중 1장을 **비밀리에** 선택
- 손패 앞면으로 뒤집어 자기 앞에 내려둠 (아직 공개 X)

**Step 2 · 동시 공개 & 배치 (Reveal & Place)**
- 모두 카운트 후 동시에 공개
- 각자 자기 타블로의 **가장 오른쪽 빈 슬롯**에 배치
  - 라운드 1: 왼쪽 1번 슬롯
  - 라운드 2: 왼쪽 2번 슬롯
  - ...
  - 라운드 8: 오른쪽 8번 슬롯
- 카드의 **보상 아이콘**이 즉시 타블로에 활성화 → 오른쪽으로 향할 미래 카드들의 조건 재료 역할

**Step 3 · 뽑기 순서 결정 (Draw Order)**
- 이번 라운드 놓은 카드의 **번호 오름차순**
- 즉 가장 **낮은 번호** 카드 놓은 사람이 **1등** (첫 뽑기)
- 가장 **높은 번호** 카드 놓은 사람이 **꼴등**
- 예: A=34, B=12, C=61 → 순서는 B(12) → A(34) → C(61)

**Step 4 · 드로우 액션 (Draw Action)**
각 플레이어가 순서대로 자기 턴에 다음 액션 수행:

**(a) 지역 카드 1장 (필수)**: 지역 시장 3장 중 1장 선택 → 손패 추가

**(b) 성소 카드 1장 (조건부 옵션)**:
- 이번 라운드 놓은 카드 번호가 **이전 라운드에 놓은 카드 번호보다 크면** → 성소 시장에서 1장 뽑을 수 있음
- 즉 **오름차순 배치**를 유지하면 매 라운드 성소 획득 가능
- 라운드 1은 이전 카드가 없으므로 성소 자동 접근 가능
- 성소는 손패 위 별도 슬롯에 보관

**Step 5 · 시장 리필 (Refill)**
- 지역 시장 3장 채움
- 성소 시장 3장 채움
- 다음 라운드로 진행

**중요**: Step 4에서 뽑기 순서가 순차적이라 후발주자는 시장에 남은 카드가 이미 줄어들어 있음. 그래서 낮은 번호 = 좋은 지역 픽, 높은 번호 = 성소 접근 · 지역은 남은 것 이라는 트레이드오프 발생.

---

### ⭐ 끝판 역방향 스코어링 (핵심 트릭)

8라운드 후 종료. 각 플레이어의 **타블로 8장을 오른쪽 끝(라운드 8 카드)부터 왼쪽 끝(라운드 1 카드)** 순으로 스캔:

**각 카드마다 3단계**:
1. 이 카드의 **조건 아이콘**을 확인
2. **이 카드보다 왼쪽에 있는 카드들**의 보상 아이콘 총합 계산 (오른쪽 카드는 무시)
3. 조건 충족 → 카드 점수 획득 · 실패 → **0점**

**핵심 함의**:
- 가장 **오른쪽 카드 (Slot 8)** = 왼쪽 7장의 아이콘을 다 활용 → 최대 스코어러 슬롯
- 가장 **왼쪽 카드 (Slot 1)** = 왼쪽에 아무것도 없음 → 조건이 있으면 자동 0점 → **셋업 (아이콘 공급) 역할만**
- 중간 슬롯 카드들 = 자기 왼쪽에 있는 카드들만큼만 자원 활용

**성소 스코어링 (별도)**
- 손패에 있던 성소 카드들의 특수 스코어링 조건 판정
- 예: `🌙 × 2` 성소 카드 → 타블로에 🌙 아이콘 5장 → +10점
- 지역 스코어와 합산

**최종 점수 = 지역 카드 점수 합계 + 성소 카드 점수 합계**
가장 높은 점수 승리.

### 전략 축 3가지

이 게임의 재미는 이 3가지 축이 매 라운드 상호작용:

**축 1 · 이번 라운드 놓는 카드 번호 (낮음 vs 높음)**
- 낮은 카드 놓기 → 뽑기 1등 → 지역 시장 best pick
- 높은 카드 놓기 → 뽑기 꼴등 → 지역은 잔반 · 다만 오름차순이면 성소 획득

**축 2 · 조건 위치 (슬롯 초반 vs 후반)**
- Slot 1~2에는 **조건 없는 카드**가 유리 (있어봐야 왼쪽 자원 없어서 0점)
- Slot 6~8에는 **조건 큰 대박 카드** 배치 (왼쪽 7장 자원 다 활용 가능)
- 중간 슬롯은 회색지대

**축 3 · 오름차순 수열 유지**
- 8라운드 내내 오름차순으로 놓으면 매 라운드 성소 획득 (+8 성소 카드 · 스코어 폭발 가능)
- 근데 큰 번호 카드는 몇 장 뿐이라 라운드 8까지 아껴야 함
- 그럼 초반에 저번호로 나서서 지역 픽은 유리하지만 대박 카드 못 뽑음

### 판단 예시 (Round 5)

**현재 상태**:
- 타블로 4장: `#14 · #22 · #29 · #35` (오름차순 유지 중)
- 타블로 활성 아이콘: 🌙🌙🌙 · 🐚🐚 · 🌲

**손패 3장**:
- `#12 매화` — 조건 `2× 🌲`, 점수 +6, 제공 ☀️🐚
- `#41 폭포` — 조건 `3× 🌙`, 점수 +15, 제공 🔮
- `#63 관측대` — 조건 없음, 점수 +5, 제공 🌙🌙🌙

**옵션 A · `#63` 놓기 (안전 스코어러 + 셋업)**
- 오름차순 유지 (35 → 63) → **성소 획득**
- 조건 없음 → 5점 확정
- 🌙 3장 추가 → 라운드 6~8에 🌙 조건 카드 오면 폭발
- ❌ 뽑기 꼴등

**옵션 B · `#41` 놓기 (즉시 대박)**
- 오름차순 유지 (35 → 41) → **성소 획득**
- 조건 `3× 🌙` 충족 → **15점 확실 획득**
- 🔮 아이콘 추가 (희귀)
- ⚠️ 41은 63보다 작아서 63을 다음에 놓으면 성소 못 뽑음 (63→41 될 순 없음)
- 중간 순서 뽑기

**옵션 C · `#12` 놓기 (희생)**
- 35 → 12 → 오름차순 깨짐 → **성소 못 뽑음**
- 조건 `2× 🌲`인데 🌲 1장뿐 → **0점**
- 🐚 추가하지만 앞선 카드에 도움 안 됨
- ✅ 뽑기 1등 → 지역 시장 best pick 확보

**분석**:
- A는 다목적 (성소 + 미래 셋업) — 3라운드 남았고 🌙 조건 카드 나올 확률 있음
- B는 확정 15점이지만 63을 다시 놓을 기회 상실 (내림차순)
- C는 손해지만 뽑기 우선권으로 매화보다 좋은 카드 얻을 가능성

**베스트 플레이는 상황 의존적**: 이런 3중 계산이 매 라운드.

## 왜 이 게임이 remake target으로 좋은가

### 원작 자체의 강점
- **엄청난 압축**: 룰 5분 · 첫 판 15분 · 마스터 20판 이상
- **재플레이성**: 68장 지역 + 54장 성소 조합으로 매 판 상황 완전 다름
- **명확한 hook**: "역방향 스코어링" 한 문장으로 게임 인상 설명 가능
- **솔로 프렌들리**: 원작에 이미 솔로 봇 룰 있음 (봇 카드 덱으로 자동 재현)
- **매크로 결정 + 마이크로 텐션**: 판당 큰 결정 8번 (라운드마다 카드 선택) + 각 결정 안에 3축 계산

### 모바일 remake와 궁합
- **세션 15분 = 완벽한 지하철 사이즈**
- **화면 세로 최적**: 타블로 좌↔우 스크롤, 손패 하단, 시장 상단
- **애니메이션이 살아남**: 스코어링 세리모니에서 오른쪽→왼쪽 하이라이트되는 순간이 압도적으로 만족스러움
- **AI 봇 이식 수월**: 원작 봇은 결정 트리 기반이라 코드로 옮기기 쉬움

### 챌린저스와 비교
| 항목 | Challengers! | Faraway |
|------|------|---------|
| 세션 | 5-10분 × 7라운드 | 15분 × 1판 |
| 카드 결정 | 자동 리빌 (플레이어 개입 0) | 매 라운드 3장 중 1장 선택 |
| 재미 축 | 벤치 관리 · 뽑기 운 | 미래 예측 · 3중 전략 계산 |
| 감성 | 지하 결투장 (긴장) | 새벽 탐험 (몰입) |
| 관전 vs 결정 | 관전 중심 (문제 지점) | 결정 중심 (해결됨) |

## 코어 루프 (한 라운드 60~90초)

각 라운드에서 플레이어가 겪는 UX 흐름:

1. **손패 검토 (10~20s)** — 손패 3장을 훑어봄. 각 카드의:
   - 번호 (뽑기 순서)
   - 조건 아이콘 · 점수
   - 제공 아이콘 (미래 셋업)

2. **타블로 상태 파악 (5~10s)** — 지금까지 놓은 카드의 활성 아이콘 확인 · 오름차순 유지 상태 확인.

3. **미래 예측 계산 (10~20s)** —
   - 이번 카드가 조건 카드면 → 지금 조건 충족 가능?
   - 셋업 카드면 → 다음 뽑기에 어떤 조건 카드가 나올까?
   - 오름차순 유지할 카드 있나?

4. **카드 놓기 (5s)** — 확정. 배치 애니메이션 (툭!)

5. **뽑기 순서 계산 (5s)** — 봇 카드와 비교. 애니메이션으로 결과 알림.

6. **드로우 (10~20s)** —
   - 지역 시장에서 1장 선택
   - 오름차순 유지 시 성소 시장에서 1장 추가 선택

**8라운드 후 → 스코어링 세리모니 (30~60s, 시그니처 씬)**:
- 타블로가 오른쪽부터 하이라이트 되며 점수 계산
- 각 카드의 조건과 왼쪽 아이콘 매칭이 시각적으로 애니메이션
- 최종 스코어 발표

## 시그니처 모멘트

### 1. 스코어링 세리모니 ★
8장 타블로가 오른쪽부터 하이라이트되며 진행되는 30~60초 씬:
- 각 카드가 스포트라이트 받음
- 조건 아이콘 확대 표시
- 왼쪽 카드들 아이콘 스캔 (실시간 카운트업)
- 조건 충족 시 → 골드 링 뜨고 점수 카운트업 · 실패 시 → 회색 fade
- 총합이 상단에 큰 Cinzel 숫자로 누적

### 2. 미래 예언 하이라이트
카드 놓기 전 프리뷰:
- 이 카드가 조건 충족 가능한지 실시간 판정
- 다른 손패의 각 카드가 지금 놓는 카드의 조건에 얼마나 도움 되는지 화살표 표시

### 3. 성소 등장
높은 숫자 놓았을 때 오름차순 유지 시 → 성소 시장 강조 애니메이션. 유리 케이스에 진열된 카드들 반짝. 하나 픽업 시 신비 챠임 + 홀로그램 발동 프리뷰.

### 4. 오름차순 트래커
현재까지 놓은 카드 번호 시퀀스를 상단에 미니 chart로 표시. 오름차순이면 골드 라인, 깨지면 그레이 라인. 시각적으로 "지금 성소 접근권을 유지 중인가" 즉시 확인 가능.

## 현 프로젝트 (챌린저스) 대비 tradeoffs

**얻는 것**
- ✅ 압도적으로 **다른 감성** (사이버펑크 → 판타지 힐링)
- ✅ 원작 룰이 정말 elegant — remake 아름다움 살아남
- ✅ 세션 15분 → 모바일 완벽
- ✅ 솔로가 원작에 이미 있음 (봇 rule 그대로 이식)
- ✅ 스코어링 세리모니가 감정 이입 훅
- ✅ 미래 예측 요소가 챌린저스 자동 리빌보다 훨씬 능동적
- ✅ 3중 전략 축 (숫자·조건·오름차순)이 두뇌 훈련
- ✅ IP 회피 상대적으로 수월 (지역·판타지는 창작 가능)

**잃는 것**
- ❌ 지금 챌린저스 프로젝트를 처음부터 다시 시작
- ❌ 아트 리소스가 훨씬 중요 (풍경 일러스트 68장)
- ❌ 밸런싱 리소스 (지역 68 + 성소 54 카드 튜닝)
- ❌ 원작 팬이 감지 못하는 미묘한 룰 스킵이 있을 수 있음
- ❌ 세리모니 씬 개발 부담
- ❌ 성소 다양한 특수 효과 시스템 구현
